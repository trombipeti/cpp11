Detailed Description

Referenciaszámlálással rendelkező stringértékekkel működő, általános szöveg osztály.

Létrehozható hagyományos C stringből, beolvasva is (pl std::cin-ről), kiírható a kimenetre, összefűzhető stringgel és karakterrel. A használónak nem kell törődnie a belső erőforrások kezelésével, az megtörténik automatikusan.
